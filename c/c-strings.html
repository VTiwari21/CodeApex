C String

A string in C is a sequence of characters stored in a character array and terminated by a null character \0. Unlike some languages, C does not have a built-in string type. Strings are widely used for handling text, input/output operations, and data processing.

Declaration and Initialization

Strings are declared as character arrays using char. They can be initialized using double quotes or by assigning characters individually. The null character \0 is automatically added at the end. Proper size allocation is important to store all characters including the terminating null character.

String Input and Output

C provides functions like scanf() and gets() (deprecated) for input, and printf() for output. For safer input, fgets() is preferred. Strings are printed using %s format specifier. Handling input properly is important to avoid buffer overflow and unexpected behavior.

String Functions

C provides built-in string handling functions in the <string.h> library. Common functions include strlen() for length, strcpy() for copying, strcat() for concatenation, and strcmp() for comparison. These functions simplify working with strings and reduce manual coding effort.

Advantages and Limitations

Strings allow efficient handling of text data in programs. They are flexible and widely used in file handling and user input. However, they require manual memory management and are prone to errors like buffer overflow if not handled carefully, making them less safe than higher-level languages.

Basic Example

#include <stdio.h>
#include <string.h>

int main() {
    char str[20] = "Hello";

    printf("String: %s\n", str);
    printf("Length: %lu\n", strlen(str));

    return 0;
}
