C Sorting

Sorting in C is done using algorithms like bubble sort, selection sort, or quick sort. Since C has no built-in sorting function, sorting must be implemented manually.

Types of Sorting

Common algorithms include bubble sort, selection sort, insertion sort, merge sort, and quick sort. Each has different time complexities and use cases.

Working of Sorting

Sorting works by comparing elements and swapping them until the array is arranged in the desired order.

Advantages and Limitations

Provides full control over implementation but requires more code and effort compared to high-level languages.

Basic Example

#include <stdio.h>

int main() {
    int arr[5] = {5, 2, 9, 1, 3};
    int i, j, temp;

    for(i = 0; i < 5; i++) {
        for(j = 0; j < 5 - i - 1; j++) {
            if(arr[j] > arr[j + 1]) {
                temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }

    for(i = 0; i < 5; i++) {
        printf("%d ", arr[i]);
    }

    return 0;
}
