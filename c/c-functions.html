C Function

A function in C is a block of code designed to perform a specific task. It helps in breaking a program into smaller, manageable parts. Functions improve code readability, reusability, and debugging by allowing the same logic to be used multiple times

Function Declaration and Definition

A function is first declared (prototype) and then defined. The declaration tells the compiler about the function’s name, return type, and parameters. The definition contains the actual code to execute. Proper declaration ensures that the function can be used before its actual implementation.

Types of Functions

Functions in C can be categorized based on parameters and return values: with arguments and return value, with arguments and no return value, without arguments but with return value, and without both. This flexibility allows functions to be used in different programming scenarios.

Function Parameters and Return Values

Functions can accept inputs called parameters and may return a value using the return statement. Parameters allow passing data into functions, while return values send results back to the caller. This makes functions useful for calculations and modular programming.

Advantages of Functions

Functions reduce code duplication and make programs easier to understand and maintain. They support modular programming, allowing different parts of a program to be developed and tested independently. Functions also improve debugging, as errors can be isolated within specific parts of the code.

Basic Example

#include <stdio.h>

int add(int a, int b);

int main() {
    int result = add(5, 3);
    printf("Sum = %d", result);
    return 0;
}

// function definition
int add(int a, int b) {
    return a + b;
}
