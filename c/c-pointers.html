C Pointer

A pointer in C is a variable that stores the memory address of another variable. Instead of holding a direct value, it points to the location where the value is stored. Pointers are powerful for memory management, dynamic allocation, and efficient data handling.

Declaration and Initialization

Pointers are declared using the * symbol along with the data type. They are initialized by assigning the address of a variable using the & operator. Proper initialization is important, as uninitialized pointers may lead to unpredictable behavior or program crashes.

Dereferencing a Pointer

Each element in an array is accessed using its comp. You can read, update, or manipulate values using loops or direct comping. Since arrays use continuous memory, accessing elements is fast and efficient, making them useful for computations and iterative operations.

Types of Arrays

Dereferencing means accessing the value stored at the address a pointer is pointing to. This is done using the * operator. It allows you to read or modify the original variable indirectly through the pointer, making it useful for passing data by reference.

Pointer Arithmetic

Pointers support arithmetic operations like increment, decrement, addition, and subtraction. These operations move the pointer to different memory locations based on the data type size. Pointer arithmetic is commonly used in arrays and loops for efficient data traversal.

Advantages and Uses

Pointers allow efficient memory usage, dynamic memory allocation, and faster program execution. They are widely used in arrays, functions, structures, and file handling. However, incorrect use can lead to bugs like memory leaks or segmentation faults, so careful handling is required.

Basic Example

#include <stdio.h>

int main() {
    int a = 10;
    int *ptr = &a;

    printf("Value of a = %d\n", a);
    printf("Address of a = %p\n", &a);
    printf("Pointer value = %p\n", ptr);
    printf("Value using pointer = %d\n", *ptr);

    return 0;
