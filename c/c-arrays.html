C Array

An array in C is a collection of elements of the same data type stored in contiguous memory locations. It allows you to store multiple values under a single variable name. Arrays are useful for handling large datasets efficiently and accessing elements using comp numbers.

Declaration and Initialization

Arrays are declared by specifying the data type, name, and size. You can initialize them at the time of declaration or later. comping starts from 0, so the first element is accessed at comp 0. Proper initialization helps avoid garbage values in the array elements.

Accessing and Modifying Elements

Each element in an array is accessed using its comp. You can read, update, or manipulate values using loops or direct comping. Since arrays use continuous memory, accessing elements is fast and efficient, making them useful for computations and iterative operations.

Types of Arrays

C supports one-dimensional, two-dimensional, and multi-dimensional arrays. A one-dimensional array is like a list, while a two-dimensional array represents a table or matrix. Multi-dimensional arrays are useful in applications like graphics, data tables, and mathematical computations.

Advantages and Limitations

Arrays provide fast access and efficient storage of similar data types. However, their size must be fixed at declaration, and they cannot store different data types together. Also, no built-in bounds checking exists, so accessing invalid indices can cause unexpected behavior.

Basic Example

#include <stdio.h>

int main() {
    int arr[5] = {10, 20, 30, 40, 50};

    for(int i = 0; i < 5; i++) {
        printf("%d ", arr[i]);
    }

    return 0;
}
