C Loop

Loops in C are used to execute a block of code repeatedly until a condition is met. They help reduce code repetition and make programs efficient. Loops are commonly used for tasks like traversing arrays, performing calculations, and handling repetitive operations.

for Loop

The for loop is used when the number of iterations is known beforehand. It consists of initialization, condition, and increment/decrement in one line. This makes it compact and easy to read, especially for counting-based operations like iterating through arrays or sequences.

while Loop

The while loop executes a block of code as long as the condition remains true. It is useful when the number of iterations is not known in advance. The condition is checked before execution, so the loop may not run at all if the condition is false initially.

do-while Loop

The do-while loop is similar to the while loop but guarantees at least one execution of the loop body. The condition is checked after executing the code. This makes it useful when the loop must run at least once regardless of the condition.

Control Statements in Loops

C provides control statements like break and continue to manage loop execution. The break statement exits the loop immediately, while continue skips the current iteration and moves to the next. These are useful for adding flexibility and control within loops.

Basic Example

#include <stdio.h>

int main() {
    int i;

    // for loop
    for(i = 1; i <= 3; i++) {
        printf("For Loop: %d\n", i);
    }

    // while loop
    i = 1;
    while(i <= 3) {
        printf("While Loop: %d\n", i);
        i++;
    }

    // do-while loop
    i = 1;
    do {
        printf("Do-While Loop: %d\n", i);
        i++;
    } while(i <= 3);

    return 0;
}
