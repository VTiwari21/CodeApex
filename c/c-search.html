C Searching

Searching in C is performed using algorithms like linear search and binary search. Since there are no built-in search functions, these algorithms must be implemented manually.

Types of Searching

Linear search checks each element one by one, while binary search works on sorted arrays and is much faster.

Working of Searching

In linear search, elements are compared sequentially until the key is found. Binary search divides the array and checks the middle element repeatedly.

Advantages and Limitations

Gives full control over implementation but requires more code. Binary search is efficient but needs sorted data.

Basic Example

#include <stdio.h>

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    int key = 30, i, found = 0;

    for(i = 0; i < 5; i++) {
        if(arr[i] == key) {
            found = 1;
            break;
        }
    }

    if(found)
        printf("Element found");
    else
        printf("Element not found");

    return 0;
}
