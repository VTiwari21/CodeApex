C# Array

An array in C# is a collection of elements of the same type stored in a fixed-size structure. Arrays are reference types and provide safe access with built-in bounds checking.

Declaration and Initialization

Arrays are declared using square brackets and can be initialized using the 'new' keyword or directly with values. All elements are automatically initialized to default values like 0 or null.

Accessing and Modifying Elements

Elements are accessed using comp numbers starting from 0. You can update values easily and use loops for iteration. C# ensures safety by throwing exceptions for invalid indices.

Types of Arrays

C# supports single-dimensional, multi-dimensional, and jagged arrays. Jagged arrays are arrays of arrays and provide more flexibility for irregular data structures.

Advantages and Limitations

Arrays are fast, type-safe, and easy to use. However, they have fixed size and cannot store different data types in a single array.

Basic Example

using System;

class Program {
    static void Main() {
        int[] arr = {10, 20, 30, 40, 50};

        for(int i = 0; i < arr.Length; i++) {
            Console.Write(arr[i] + " ");
        }
    }
}
