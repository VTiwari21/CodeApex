C# Sorting

Sorting in C# is done using built-in methods like Array.Sort() which provides efficient sorting of elements.

Types of Sorting

C# uses introspective sort, combining quicksort, heapsort, and insertion sort.

Working of Sorting

Sorting is performed using Array.Sort() which arranges elements in ascending order.

Advantages and Limitations

Simple and efficient with built-in methods, but limited control over sorting logic.

Basic Example

using System;

class Program {
    static void Main() {
        int[] arr = {5, 2, 9, 1, 3};

        Array.Sort(arr);

        foreach(int i in arr) {
            Console.Write(i + " ");
        }
    }
}
