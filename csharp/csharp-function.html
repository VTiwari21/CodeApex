C# Function

A function in C# (also called a method) is a block of code that performs a specific task. It helps divide a program into smaller parts, improving readability and organization. Functions allow code reuse, making programs more efficient and easier to maintain

Function Declaration and Initialization

In C#, functions are defined inside a class using a return type, method name, and parameters. There is no separate declaration like in C. The method signature specifies how the function is called, and the body contains the code that executes when invoked.

Types of Functions

C# supports different types of functions such as methods with parameters and return values, methods without parameters, and methods without return values. It also supports advanced features like method overloading, recursion, and static methods for flexible programming.

Parameters and Return Values/h2>

Functions can accept parameters to receive input and return values using the return statement. C# also supports passing arguments by value and by reference using ref and out keywords, allowing efficient data handling and modification within methods.

Advantages of Functions

Functions improve code modularity, reusability, and readability. They make debugging easier by isolating logic into smaller parts. Functions also support teamwork, as different developers can work on separate methods, making large projects easier to develop and maintain.

Basic Example

using System;

class Program {

    // function (method)
    static int Add(int a, int b) {
        return a + b;
    }

    static void Main() {
        int result = Add(5, 3);
        Console.WriteLine("Sum = " + result);
    }
}
