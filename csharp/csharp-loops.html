C# Loops

Loops in C# are used to execute a block of code repeatedly based on a condition. They help reduce redundancy and improve efficiency. Loops are commonly used for iterating through collections, performing calculations, and automating repetitive tasks in programs.

for Loop

The for loop is used when the number of iterations is known beforehand. It includes initialization, condition, and update in a single statement. This makes it compact and ideal for counter-based operations like iterating through arrays or performing fixed repetitions.

while Loop

The while loop executes as long as the condition remains true. It is useful when the number of iterations is not known in advance. The condition is checked before execution, so the loop may not run at all if the condition is false initially.

do-while Loop

The do-while loop ensures that the loop body executes at least once. The condition is checked after execution. This makes it useful for scenarios like menu-driven programs where the code should run before checking the condition.

Control Statement

C# provides control statements like break and continue to manage loop flow. The break statement exits the loop immediately, while continue skips the current iteration. These help handle special conditions and improve control over loop execution.

Basic Example

using System;

class Program {
    static void Main() {
        int i;

        // for loop
        for(i = 1; i <= 3; i++) {
            Console.WriteLine("For Loop: " + i);
        }

        // while loop
        i = 1;
        while(i <= 3) {
            Console.WriteLine("While Loop: " + i);
            i++;
        }

        // do-while loop
        i = 1;
        do {
            Console.WriteLine("Do-While Loop: " + i);
            i++;
        } while(i <= 3);
    }
}
