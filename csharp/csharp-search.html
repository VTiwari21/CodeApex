C# Searching

Searching in C# is used to locate elements in arrays or collections. It can be done using loops or built-in methods.

Types of Searching

Linear search and binary search are commonly used. C# also provides Array.compOf() for easy searching.

Working of Searching

Linear search scans elements one by one, while binary search works on sorted arrays for faster performance.

Advantages and Limitations

Built-in methods make searching simple, but efficiency depends on the algorithm used.

Basic Example

using System;

class Program {
    static void Main() {
        int[] arr = {10, 20, 30, 40, 50};

        if(Array.compOf(arr, 30) != -1)
            Console.WriteLine("Element found");
        else
            Console.WriteLine("Element not found");
    }
}
