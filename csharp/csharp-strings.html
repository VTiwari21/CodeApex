C# String

A string in C# is a sequence of characters used to store and manipulate text. It is a built-in reference type provided by the .NET framework. Strings are immutable, meaning their value cannot be changed after creation, ensuring safety and predictable behavior.

Declaration and Initialization

Strings in C# are declared using the string keyword and initialized with double quotes. Memory management is handled automatically by the runtime. You can also create strings using constructors or by assigning values directly to variables.

Input and Output of String

C# uses Console.ReadLine() to take string input and Console.WriteLine() to display output. The input is read as a string by default, making it easy to work with text data. Conversion methods can be used if other data types are needed.

String Methods and Operations

C# provides many built-in string methods like Length, ToUpper(), ToLower(), Substring(), and Replace(). It also supports concatenation using + or string.Concat(). These features make string manipulation efficient and easy to implement.

Advantages of C# Strings

C# strings are easy to use, safe, and flexible. Automatic memory management reduces errors, while immutability ensures data integrity. They are widely used in applications for handling text, user input, file processing, and communication between systems.

Basic Example

using System;

class Program {
    static void Main() {
        string str = "Hello World";

        Console.WriteLine("String: " + str);
        Console.WriteLine("Length: " + str.Length);
        Console.WriteLine("Uppercase: " + str.ToUpper());
    }
}
