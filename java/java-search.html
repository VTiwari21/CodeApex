Java Searching

Searching in Java is used to find elements in arrays or collections. It can be done manually or using built-in methods.

Types of Searching

Linear search and binary search are commonly used. Java provides Arrays.binarySearch() for fast searching.

Working of Searching

Linear search checks elements one by one. Binary search works on sorted arrays and reduces search time significantly.

Advantages and Limitations

Easy to implement with built-in methods, but binary search requires sorted data.

Basic Example

public class Main {
    public static void main(String[] args) {
        int[] arr = {10, 20, 30, 40, 50};
        int key = 30;
        boolean found = false;

        for(int i : arr) {
            if(i == key) {
                found = true;
                break;
            }
        }

        if(found)
            System.out.println("Element found");
        else
            System.out.println("Element not found");
    }
}
