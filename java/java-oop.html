Java OOP

Object-Oriented Programming (OOP) in Java is based on classes and objects. It helps organize programs using real-world concepts. OOP improves code structure, reusability, and scalability, making it suitable for large applications.

Classes and Objects

A class is a blueprint that defines properties and methods, while an object is an instance of a class. Objects use class definitions to perform tasks, enabling modular and organized programming.

Encapsulation

Encapsulation is the process of wrapping data and methods into a single unit. It restricts access using access modifiers like private and public. This ensures data security and prevents unintended modification.

Inheritance and Polymorphism

Inheritance allows one class to inherit features from another, promoting reuse. Polymorphism allows methods to behave differently based on context. These concepts make programs flexible and extensible.

Advantages of OOP in Java

OOP improves modularity, maintainability, and scalability. It helps manage complex systems by dividing them into smaller parts. It also enhances code reuse and security, making it widely used in enterprise applications.

Basic Example

class Student {
    String name;

    void display() {
        System.out.println("Name: " + name);
    }
}

public class Main {
    public static void main(String[] args) {
        Student s1 = new Student();
        s1.name = "Vaibhav";
        s1.display();
    }
}
