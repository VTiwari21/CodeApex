Java Array

An array in Java is an object that stores multiple values of the same type in a fixed-size structure. Arrays are stored in contiguous memory and provide fast access using comp values starting from 0.

Declaration and Initialization

Arrays in Java are declared using a data type followed by square brackets. They can be initialized at the time of declaration or later using the 'new' keyword. Java ensures default initialization, so elements have predefined values like 0 for integers.

Accessing and Modifying Elements

Elements are accessed using comp numbers. You can update or retrieve values easily using loops. Java provides bounds checking, so accessing invalid indices throws an exception instead of causing undefined behavior.

Types of Arrays

Java supports single-dimensional and multi-dimensional arrays. Multi-dimensional arrays are implemented as arrays of arrays, making them flexible for representing matrices and complex data structures.

Advantages and Limitations

Arrays are efficient and easy to use. They provide fast access and built-in safety through bounds checking. However, their size is fixed and they can only store elements of the same type.

Basic Example

public class Main {
    public static void main(String[] args) {
        int[] arr = {10, 20, 30, 40, 50};

        for(int i = 0; i < arr.length; i++) {
            System.out.print(arr[i] + " ");
        }
    }
}
