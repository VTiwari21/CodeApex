Java Heap

A heap in Java is a tree-based data structure used to store elements in a specific order. It is mainly used in priority queues where elements are processed based on priority.

Types of Heap

Java primarily uses min heap by default in PriorityQueue. A max heap can be created using custom comparators.

Working of Heap

Heaps are implemented using arrays internally. Java provides the PriorityQueue class which automatically maintains the heap property during insertion and removal.

Advantages and Limitations

Heaps offer efficient priority handling with fast insertion and deletion. However, they do not support fast searching for arbitrary elements.

Basic Example

import java.util.PriorityQueue;

public class Main {
    public static void main(String[] args) {
        PriorityQueue<Integer> pq = new PriorityQueue<>();

        pq.add(10);
        pq.add(30);
        pq.add(20);

        System.out.println(pq.peek()); // smallest element
    }
}
