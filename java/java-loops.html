Java Loops

Java is object-oriented, secure, and robust. It provides automatic memory management through garbage collection. Java supports multithreading and has a rich standard library. Its platform independence (“write once, run anywhere”) makes it highly popular for cross-platform application development.

for Loop

Loops in Java are used to execute a block of code repeatedly based on a condition. They help reduce redundancy and improve efficiency. Loops are commonly used for iterating through arrays, performing calculations, and automating repetitive tasks.

while Loop

The while loop executes as long as a condition remains true. It is useful when the number of iterations is not known beforehand. The condition is checked before execution, so the loop may not run if the condition is false initially.

do-while Loop

The do-while loop ensures that the loop body executes at least once. The condition is checked after execution. It is useful in cases like menu-driven programs where execution must happen before checking the condition.

Control Statements in Loops

Java provides break and continue to control loop execution. The break statement exits the loop, while continue skips the current iteration. These help in managing flow and handling special conditions inside loops.

Basic Example

public class Main {
    public static void main(String[] args) {
        int i;

        // for loop
        for(i = 1; i <= 3; i++) {
            System.out.println("For Loop: " + i);
        }

        // while loop
        i = 1;
        while(i <= 3) {
            System.out.println("While Loop: " + i);
            i++;
        }

        // do-while loop
        i = 1;
        do {
            System.out.println("Do-While Loop: " + i);
            i++;
        } while(i <= 3);
    }
}
