Java String

A string in Java is a sequence of characters used to store text. It is an object of the String class. Strings are immutable, meaning their values cannot be changed after creation, ensuring safety and consistency.

Declaration and Initialization

Strings are declared using the String class and initialized with double quotes. Java automatically handles memory allocation. Strings can also be created using the new keyword, but direct assignment is more commonly used.

Input and Output of Strings

Strings are printed using System.out.println(). For input, Java uses the Scanner class. This allows easy handling of text input and output in programs, making user interaction simple

String Methods and Operations

Java provides methods like length(), toUpperCase(), toLowerCase(), substring(), and equals(). These help in manipulating and comparing strings efficiently without manual handling.

Advantages of Java Strings

Java strings are easy to use, secure, and come with many built-in methods. Their immutability ensures data safety. They are widely used in applications for text processing, file handling, and user interaction.

Basic Example

public class Main {
    public static void main(String[] args) {
        String str = "Hello World";

        System.out.println("String: " + str);
        System.out.println("Length: " + str.length());
        System.out.println("Uppercase: " + str.toUpperCase());
    }
}
