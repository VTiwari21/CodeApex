Pyhton String

A string in Python is a sequence of characters used to represent text. It is enclosed in single, double, or triple quotes. Strings are immutable, meaning their content cannot be changed after creation, making them safe and predictable for handling textual data.

Declaration and Initialization

Strings are created by assigning text to a variable using quotes. Python allows single (' '), double (" "), and triple quotes (''' ''' or """ """) for multi-line strings. No need to define size or type explicitly, as Python handles memory automatically.

Input and Output of Strings

Python uses input() to take string input from the user and print() to display output. The input() function returns data as a string by default. This makes string handling simple and straightforward, especially for user interaction in programs.

String Operations and Functions

Python provides many built-in string methods like len(), lower(), upper(), strip(), and replace(). Strings also support slicing and comping for accessing parts of the string. These features make string manipulation powerful and easy to implement.

Advantages of Python Strings

Python strings are easy to use, flexible, and come with many built-in functions. They handle memory automatically and reduce errors. Their immutability ensures data safety, and they are widely used in text processing, file handling, and data analysis tasks.

Basic Example

str1 = "Hello World"

print("String:", str1)
print("Length:", len(str1))
print("Uppercase:", str1.upper())
