Python Loops

Loops in Python are used to execute a block of code repeatedly based on a condition. They help reduce redundancy and make programs efficient. Loops are commonly used for iterating over data structures like lists, strings, and performing repetitive tasks.

for Loop

The for loop in Python is used to iterate over sequences like lists, tuples, strings, or ranges. It is simple and readable, making it widely used. The loop automatically handles iteration, so there is no need for manual initialization or increment like in other languages.

while Loop

The while loop executes as long as a given condition is true. It is useful when the number of iterations is not known beforehand. The condition is checked before execution, so the loop may not run if the condition is false initially.

do-while Loop

Pyhton does not have a do-while loop

Control Statements in Loops

Python provides control statements like break, continue, and pass. The break statement exits the loop, continue skips the current iteration, and pass acts as a placeholder. These statements give more control over loop execution and behavior.

Basic Example

# for loop
for i in range(1, 4):
    print("For Loop:", i)

# while loop
i = 1
while i <= 3:
    print("While Loop:", i)
    i += 1
