Python Sorting

Sorting in Python is used to arrange elements in ascending or descending order. Python provides powerful built-in functions like sort() and sorted() that make sorting fast and easy.

Types of Sorting

Python uses Timsort, a hybrid sorting algorithm derived from merge sort and insertion sort. It is stable and efficient for real-world data.

Working of Sorting

Sorting can be done in-place using sort() or by creating a new sorted list using sorted(). You can also customize sorting using key functions and reverse order.

Advantages and Limitations

Python sorting is simple and efficient with minimal code. However, less control is given compared to manual implementations in lower-level languages.

Basic Example

arr = [5, 2, 9, 1, 3]

arr.sort()

print(arr)
