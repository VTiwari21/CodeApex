C++ OOP

Object-Oriented Programming (OOP) in C++ is a programming paradigm based on objects and classes. It allows you to structure programs using real-world concepts. OOP helps in organizing code efficiently, improving readability, and making programs easier to maintain and extend.

Classes and Objects

A class is a blueprint for creating objects, while an object is an instance of a class. Classes define properties (variables) and behaviors (functions). Objects use these properties and behaviors to perform tasks, enabling better organization and modular design in programs.

Accessing and Modifying Elements

Encapsulation is the concept of bundling data and methods together within a class. It restricts direct access to some components using access modifiers like private, protected, and public. This improves data security and prevents unintended interference from other parts of the program.

Inheritance and Polymorphism

Inheritance allows one class to acquire properties and behaviors of another class, promoting code reuse. Polymorphism enables functions or operators to behave differently based on context. These concepts make programs flexible, scalable, and easier to extend with new features.

Advantages of OOP

OOP improves code reusability, scalability, and maintainability. It helps in managing large and complex programs by dividing them into smaller objects. OOP also enhances data security and supports real-world modeling, making it widely used in software development.

Basic Example

#include <iostream> 
using namespace std;

class Student {
public:
    string name;

    void display() {
        cout << "Name: " << name;
    }
};

int main() {
    Student s1;
    s1.name = "Vaibhav";
    s1.display();
    return 0;
}
