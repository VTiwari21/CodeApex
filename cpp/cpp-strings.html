C++ String

A string in C++ is a sequence of characters used to store and manipulate text. Unlike C, C++ provides a built-in string class in the library. It makes handling text easier, safer, and more flexible compared to character arrays.

Declaration and Initialization

Strings in C++ are declared using the string data type. They can be initialized using double quotes. Unlike C-style strings, you do not need to worry about null characters or fixed sizes, as the string class handles memory automatically.

Input and Output of Strings

C++ provides cin and getline() for input, and cout for output. While cin reads input until a space, getline() reads an entire line including spaces. This makes string input more flexible and user-friendly compared to traditional methods.

String Functions and Operations

The string class supports many built-in functions like length(), append(), substr(), and find(). It also allows operators like + for concatenation. These features simplify string manipulation and reduce the need for manual handling of character arrays.

Advantages of C++ Strings

C++ strings are dynamic, safe, and easy to use. They automatically manage memory and reduce errors like buffer overflow. Compared to C-style strings, they provide more functionality and flexibility, making them ideal for modern programming and text processing tasks.

Basic Example

#include <siostream>
#include <string.h>

using namespace std;

int main() {
    string str = "Hello";

    cout << "String: " << str << endl;
    cout << "Length: " << str.length() << endl;

    return 0;
}
