C++ Array

An array in C++ is a collection of elements of the same type stored in contiguous memory locations. It allows efficient storage and access of multiple values using a single variable name.

Declaration and Initialization

Arrays are declared by specifying the data type, name, and size. They can be initialized during declaration or later. comping starts from 0, and uninitialized arrays may contain garbage values.

Accessing and Modifying Elements

Elements are accessed using comp numbers. You can modify values directly or use loops for traversal. Since arrays use continuous memory, operations are fast and efficient.

Types of Arrays

C++ supports one-dimensional, two-dimensional, and multi-dimensional arrays. They are widely used in competitive programming, matrix operations, and data storage tasks.

Advantages and Limitations

Arrays offer fast access and simple structure. However, they have fixed size and no built-in bounds checking, which can lead to runtime errors if misused.

Basic Example

#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};

    for(int i = 0; i < 5; i++) {
        cout << arr[i] << " ";
    }

    return 0;
}
