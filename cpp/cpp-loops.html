C++ Loops

Loops in C++ are used to execute a block of code repeatedly based on a condition. They help reduce redundancy and make programs efficient. Loops are commonly used for tasks like iterating through arrays, performing calculations, and handling repetitive operations.

for Loop

Loops in C++ are used to execute a block of code repeatedly based on a condition. They help reduce redundancy and make programs efficient. Loops are commonly used for tasks like iterating through arrays, performing calculations, and handling repetitive operations.

for Loop

The for loop is used when the number of iterations is known. It includes initialization, condition, and update in a single line. This makes it compact and easy to manage, especially when working with counters or iterating over arrays and sequences.

while Loop

The while loop runs as long as a specified condition is true. It is useful when the number of iterations is not fixed. The condition is checked before execution, so if it is false initially, the loop body will not execute even once.

do-while Loop

The do-while loop ensures that the loop body executes at least once. The condition is checked after execution. This loop is useful when you need the code to run before checking the condition, such as in menu-driven programs.

Control Statements in Loops

C++ provides control statements like break and continue to control loop execution. The break statement exits the loop immediately, while continue skips the current iteration. These help in managing loop flow and handling specific conditions effectively.

Basic Example

#include <iostream>

using namespace std;

int main() {
    int i;

    // for loop
    for(i = 1; i <= 3; i++) {
        cout << "For Loop: " << i << endl;
    }

    // while loop
    i = 1;
    while(i <= 3) {
        cout << "While Loop: " << i << endl;
        i++;
    }

    // do-while loop
    i = 1;
    do {
        cout << "Do-While Loop: " << i << endl;
        i++;
    } while(i <= 3);

    return 0;
}
