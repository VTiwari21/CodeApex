C++ Pointer

A pointer in C++ is a variable that stores the memory address of another variable. It allows direct access to memory and is widely used for dynamic memory allocation, efficient array handling, and implementing complex data structures.

Declaration and Initialization

Pointers are declared using the asterisk (*) symbol. They are initialized by assigning the address of a variable using the address-of (&) operator. Proper initialization is important to avoid undefined behavior caused by wild pointers.

Accessing and Modifying Elements

The value stored at the memory address can be accessed using the dereference (*) operator. You can modify the original variable through the pointer, making pointers powerful for passing arguments and working with memory directly.

Types of Pointers

C++ supports different types of pointers such as null pointers, void pointers, and pointer to pointer. Smart pointers like unique_ptr and shared_ptr are also available in modern C++ for safer memory management.

Advantages and Limitations

Pointers provide powerful control over memory and improve performance. However, improper use can lead to memory leaks, dangling pointers, and crashes, making them harder to use compared to high-level constructs.

Basic Example

#include <iostream>
using namespace std;

int main() {
    int x = 10;
    int* ptr = &x;

    cout << "Value of x: " << x << endl;
    cout << "Address of x: " << ptr << endl;
    cout << "Value using pointer: " << *ptr << endl;

    return 0;
}
