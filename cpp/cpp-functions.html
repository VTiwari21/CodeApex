C++ Functions

A function in C++ is a reusable block of code that performs a specific task. It helps divide a program into smaller parts, making it easier to read and manage. Functions improve code reusability and make debugging simpler by isolating logic into separate sections.

Declaration and Initialization

Functions are declared using a prototype before they are used and defined later with actual code. The declaration specifies the return type, function name, and parameters. The definition contains the body of the function, which executes when the function is called.

Types of Functions

C++ supports different types of functions such as functions with parameters and return values, functions without parameters, and functions without return values. It also supports advanced types like inline functions, recursive functions, and function overloading for flexibility.

Parameters and Return Values

Functions can accept inputs through parameters and return results using the return statement. C++ also allows passing arguments by value or by reference, enabling efficient data handling. Return values help send processed data back to the calling function.

Advantages of Functions

Functions make programs modular, reusable, and easier to maintain. They reduce code duplication and improve readability. Functions also allow team collaboration, as different parts of a program can be developed independently, making large projects more manageable.

Basic Example

#include <iostream>

using namespace std;

// function declaration
int add(int a, int b);

int main() {
    int result = add(4, 6);
    cout << "Sum = " << result;
    return 0;
}

// function definition
int add(int a, int b) {
    return a + b;
}
